AWS Lambda + S3 Automation

Processamento automático ao subir arquivos no S3, com logs e métricas no CloudWatch.

Contexto

Eliminar tarefas manuais pós-upload (validar, renomear, mover e registrar eventos). Eventos do Amazon S3 disparam uma AWS Lambda em Python para padronizar o fluxo e registrar evidências no CloudWatch.

Arquitetura

  • Amazon S3 (evento ObjectCreated)
  • AWS Lambda (Python 3.x) com IAM mínimo (least privilege)
  • CloudWatch (logs e métrica customizada)
  • Infra e função versionadas no GitHub

Fluxo

  1. Upload em s3://bucket/input/.
  2. S3 aciona a Lambda com o evento do objeto.
  3. Lambda valida e aplica a regra de negócio.
  4. Move para processed/ ou rejected/.
  5. CloudWatch registra logs e métricas para alertas.

Resultados

  • Zero intervenção manual (↑ produtividade)
  • Rastreabilidade via logs e métricas
  • Custo sob demanda (arquitetura serverless)
